# UniCrypt

Universal digital credential verification and secure reuse platform.

## Features
- **Magic Link Auth**: Instant role-based passwordless access.
- **Verification Vault**: Secure user uploads and automated status auditing.
- **Smart AI Copilot**: Production-grade ChatGPT style assistant with live web search and intent classification.
- **Superadmin Panel**: Mass status controls, user suspension overrides, and full platform logs.

## Setup
```bash
npm install
npm run dev
```
