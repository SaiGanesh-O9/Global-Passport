# UniCrypt® Brand Language & Vocabulary Guide

This document defines the official terminology, feature naming conventions, and user-facing copy guidelines to maintain platform consistency.

---

## 1. Product Suite Hierarchy

### Platform Branding
- **UniCrypt®**: The universal credential intelligence platform.

### Operating System
- **UniCrypt OS™**: The intelligent workspace context panel.

### Intelligence Layer
- **UniCrypt Match™**: Eligibility and requirements comparison mapping.
- **UniCrypt Vision™**: Page-straightening, contrast enhancement, and OCR metadata highlight tool.
- **UniCrypt Insight™**: Structured recommendation modules (Summary, Analysis, Recommendation, Risk, Actions).
- **UniCrypt Pulse™**: Live updates detailing digital twin changes.

### Infrastructure Layer
- **UniCrypt Vault™**: Secure digital vault container.
- **UniCrypt Verify™**: Verification engine matching registries.
- **UniCrypt Timeline™**: Verification status horizontal milestone maps.
- **UniCrypt Shield™**: Security, privacy, and consent controls.

## 2. Navigation Vocabulary Mapping

Use the following standard terms in navigation menus, button links, headers, and tooltips:

| Legacy Term | Rebranded Term | Description |
|---|---|---|
| Dashboard | **My Workspace** | Main workspace dashboard interface. |
| Documents / Vault | **Credential Vault** | Digital credential storage area. |
| Requests / Verification Requests | **Active Verifications** | Requested credentials matching verifiers. |
| Organizations / Partners | **Organizations** | Trust partner listing directory. |

## 3. UI Copy Constraints

### No Technical Jargon
Never expose internal engineering vocabulary to the user:
- ❌ Avoid: *OCR*, *JSON*, *Embeddings*, *Vector search*, *LLM*, *Context window*, *Metadata parsing*.
- ✔ Use: *Analyzing credential...*, *Comparing requirements...*, *Checking authenticity...*, *Preparing recommendations...*.
